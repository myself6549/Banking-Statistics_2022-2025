# 🏦 Bank-wise ATM, POS & Card Statistics Dashboard

## 📌 Project Overview

The **Bank-wise ATM, POS & Card Statistics Dashboard** is an interactive data visualization application built using **Streamlit** and **Plotly**. The dashboard analyzes banking infrastructure and digital payment statistics across Indian banks from 2022 onwards.

This project provides insights into ATM deployment, POS terminals, debit cards, credit cards, and transaction trends, helping users understand the growth of digital banking and payment systems.

---

## 🎯 Objectives

* Analyze banking infrastructure across banks.
* Compare ATM and POS deployment.
* Examine debit and credit card distribution.
* Track digital transaction trends.
* Identify top-performing banks.
* Provide interactive visual analytics for decision-making.
